container

This section contains 2 functions for building and displaying SVG containers. Each map built with the geoviz library will start with the init function and end with the render function.

container
Static Members
▸ create(options)
src/container/create.js

The create function is the first step in map construction. It creates an svg container into which the various layers can be added.

create(options: object): SVGSVGElement
Parameters
options (object = {}) options and parameters
Name Description
options.height number (default null) height of the container
options.projection function (default d3.geoEquirectangular()) projection definition. See d3-geo & d3-geo-projection
options.domain (object | Array<object>) the domain corresponds to the geographical area to be displayed. It is defined by a geoJSON or an array containing geoJSONs. By default, the entire world is represented.
options.pos Array<number> (default [0,0]) position of the container (if contained in another svg container)
options.background string (default "none") background color
options.width number (default 1000) width of the container. This value is automatically calculated according to domain . But it can be forced by entering a value.
options.margin (number | Array<number>) (default [0,0,0,0]) margins around the map. A number to set the same margin everywhere or an array [top, right, bottom, left] to set different margins.
options.parent object (default null) name of parent container into which this child container is to be included. In this case, the options.pos parameter is also used.
options.fontFamily string (default "Arial") font-family for the entire map
options.zoomable (boolean | number | string) (default false) activates the map zoom function. If you set an array of 2 values, it defines the scaleExtent (default: [1,8]). Use "versor" to activate versor zoom . "Versor" is only available for vector geometries in wgs84.
Returns
SVGSVGElement: the function returns a svg container + some information about this container: projection , margin , width , height and bbox
Example
let main = geoviz.container.create({width: 500, background: "lightblue"})

render(svg: SVGSVGElement, options: object): SVGSVGElement
Parameters

Name Description

Returns
SVGSVGElement: a pretty map in SVG format :-)
Example
